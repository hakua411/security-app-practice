
## security-app-practice

## 概要
COACHTECH 教材 Tutorial 10-6「Webセキュリティ ハンズオン」で作成した成果物です。

## 使用技術
- PHP 8.x
- Laravel 10.x
- CSRF保護(`@csrf`)、XSS対策(Bladeのエスケープ)

## 学んだこと
- コントローラーでのバリデーションルールの指定
- CSRF保護(`@csrf`ディレクティブの記述)とXSS対策(`{{}}`構文)の重要性
- ルーティングの設定

## 動作確認
1. フォームに入力して送信
2. 送信完了画面で入力内容が正しく表示されるか確認
3. `@csrf`を削除後の送信で419PageExpiredエラーになるか確認
4. メッセージ欄にスクリプトタグを入力しても実行されず文字列として表示されるか確認

## 詰まったポイントと解決方法
- バリデーションルールの設定
  →教材「9-6-1:バリデーションの基礎」を参照
- ルーティングの設定
  →教材「9-1-4:ルーティングの基礎」を参照
